# Proposal: ibabs-notubiz-connector

## Summary
Implement connectors for iBabs and NotuBiz raadsinformatiesystemen (council information systems) to synchronize meeting agendas, decisions, and documents with OpenRegister.

## Motivation
Dutch municipalities use iBabs or NotuBiz for council meeting management. Synchronizing this data with OpenRegister enables cross-referencing council decisions with cases and documents.

## Scope
- iBabs API integration (meetings, agendas, documents)
- NotuBiz API integration (meetings, decisions, attachments)
- Parafering (approval) workflow support
- Bidirectional synchronization of meeting data
